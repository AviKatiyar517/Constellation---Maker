\# 🌟 Constellation Maker



An interactive night sky canvas where you can create your own constellations.



\## ✨ Features

\- Click anywhere to place stars

\- Connect stars with lines to form constellations

\- Name and save your constellations

\- Twinkling background stars

\- Right-click to delete a star



\## 🚀 Live Demo

\[Click here to try it](https://github.com/AviKatiyar517/Constellation---Maker)



\## 🛠️ Built With

\- HTML

\- CSS

\- JavaScript (Canvas API)



\## 📖 How to Use

1\. Click on the canvas to place stars

2\. Switch to \*\*Connect\*\* mode to draw lines between stars

3\. Click \*\*Save\*\* to name your constellation

4\. View your collection anytime
